# Loan Default Prediction using PySpark

This project predicts whether a customer will default on a loan using Python and PySpark.  
It uses data preprocessing techniques and a Logistic Regression model for classification.

## Features
- Data cleaning and preprocessing
- Handling missing values
- Categorical encoding
- Logistic Regression model
- Model evaluation using AUC

## Technologies Used
- Python
- PySpark
- Spark MLlib
- Matplotlib

## Dataset
The dataset contains customer loan details such as:
- Income
- Age
- Loan amount
- Credit history
- Employment status
- Default status (target variable)

## Project Workflow
1. Load dataset using PySpark
2. Handle missing values
3. Encode categorical features
4. Assemble feature vectors
5. Train Logistic Regression model
6. Evaluate model using AUC score
